A Large Scale CAD Model Dataset

Dataset

The Large Scale CAD Model Dataset (LSC Dataset) proposes a collection of CAD structures with large dimension that can be used to improve the research of geometric deep learning methods and applications that demand such a kind of structures.

We provide a dataset with models with parametrized curves and surfaces, ground truth for differencial quantities, patch segmentation, geometric feature detection and shape reconstruction.
